# sample-todo-app

- server
lint: 
`yarn server:lint`
 or 
`yarn workspace server lint`

build:
`yarn server:build`
or
`yarn workspace build`

dev-server:
`yarn server:serve`
or
`yarn workspace server serve`


- client:
lint: 
`yarn cl:lint`
or
`yarn workspace client lint`

build:
`yarn cl:build`

`yarn workspace client build`

dev-server:
`yarn cl:dev`
or
`yarn workspace client dev`
